# Resume PDF fonts

Full **latin** TTF files for PDF rendering (Oswald, Roboto Condensed, Source Code Pro).

Do **not** use fonts extracted from the sample PDF — those are subsets and will render missing glyphs as squares (é, —, •, etc.).

Downloaded from [google-webfonts-helper](https://gwfh.mranftl.com/fonts). All OFL licensed.

Regenerate: `python3 scripts/generate_resume.py`
